## üö® Crisis Management

### Common Issues

**Major Bug:**
1. Acknowledge immediately
2. Communicate fix timeline
3. Ship update ASAP
4. Post-mortem and prevention plan

**Negative Viral Review:**
1. Respond professionally
2. Offer to make it right
3. Fix if valid criticism
4. Show the work publicly

**Community Conflict:**
1. Enforce guidelines consistently
2. Resolve privately when possible
3. Document patterns
4. Update guidelines as needed

**Competitor Criticism:**
1. Don't engage publicly
2. Focus on our strengths
3. Take high road
4. Learn from valid points
