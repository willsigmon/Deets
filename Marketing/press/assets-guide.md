## 2. App Icon & Logos

### App Icon

**Sizes Available:**
- `icon-1024.png` - 1024×1024px (App Store)
- `icon-512.png` - 512×512px (Web use)
- `icon-256.png` - 256×256px (Smaller web use)
- `icon-180.png` - 180×180px (iOS Home Screen @3x)
- `icon-120.png` - 120×120px (iOS Home Screen @2x)
- `icon-60.png` - 60×60px (iOS Home Screen @1x)

**Formats:**
- PNG (transparent background)
- PNG (with background color: #23C4AE)
- SVG (vector, scalable)

### Company Logo

**Variants:**
- `logo-color.svg` - Full color version
- `logo-white.svg` - White for dark backgrounds
- `logo-black.svg` - Black for light backgrounds
- `logo-horizontal.svg` - Horizontal layout with text
- `logo-square.svg` - Square layout

**Sizes:** Vector (SVG) + PNG exports in 512px, 1024px, 2048px

### Logo Usage Guidelines

**DO:**
- Maintain original proportions
- Use adequate clear space (minimum = height of icon)
- Use provided color variations

**DON'T:**
- Stretch or distort
- Change colors
- Add effects or shadows
- Rotate or skew
- Place on busy backgrounds

**Clear Space:** Minimum clear space around logo should equal the height of the app icon.

**Minimum Size:**
- Digital: 24px height
- Print: 0.5 inches height
